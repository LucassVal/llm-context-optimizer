> **"O objetivo não é uma IA mais inteligente. É uma IA que se lembra."**

O NeoCortex é um framework de orquestração localizado SOTA (State-of-The-Art), projetado para unir o raciocínio de LLMs aos fluxos de trabalho industriais profissionais. Utilizando uma **Arquitetura de Memória Fractal** via o Model Context Protocol (MCP), o NeoCortex elimina a amnésia de contexto e reduz drasticamente os custos operacionais de APIs.

---

## 🚀 Por que NeoCortex?

Agentes de IA tradicionais sofrem com o esvaziamento constante de contexto e consumo redundante de tokens. O NeoCortex resolve isso através de lobos de memória persistentes de alta performance.

## 🧠 Recursos Core SOTA

- **Governança Zero-Trust:** IAM criptográfico integrado para execução segura de ferramentas.
- **IPC vía Memória Compartilhada:** Mapeamento de memória RAM (mmap) de alta velocidade para transferência de payloads entre agentes.
- **Lobos Fractais:** Memória persistente hierárquica que armazena contexto de arquitetura, segurança e desenvolvimento entre sessões.
- **Decaimento Térmico (TTL):** Poda inteligente de contexto que arquiva memórias "frias" para manter a performance máxima da LLM.

## 🛡️ Auditoria Técnica e Conformidade (21/04/2026)

O NeoCortex passou por uma auditoria arquitetural completa contra as **Regras de Governança NC-GOV-FR-003** e o **Padrão de Protocolo MCP**:

- **Verificação Core:** O baseline do MVP em Python está 100% funcional com **17 módulos de ferramentas de alto nível** integrados fisicamente, fornecendo mais de **38 ações industriais únicas**.
- **Validação SOTA:**
  - `mmap IPC`: Latência O(1) verificada para payloads >1024 bytes.
  - `Thermal TTL`: Poda de contexto baseada em recência validada em `neocortex_context`.
  - `Zero-Trust`: Interceptores criptográficos ativos em todo o gateway FastMCP.
- **Portabilidade:** Padronizado em `127.0.0.1` IPv4 loopback para garantir conectividade zero-config em ambientes Windows/WSL2.

## 🛠️ Stack Técnico

- **Core:** Python 3.12 (MVP) → Em transição para **Rust** (`neocortex-mcp-rs`).
- **Transporte:** FastMCP (Padrão Anthropic).
- **Persistência:** RocksDB + SQLite FTS5.
- **Motor Vetorial:** LanceDB (Otimizado HNSW / IVFPQ).
