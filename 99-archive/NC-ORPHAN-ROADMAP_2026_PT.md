# NeoCortex: Roadmap Estratégico 2026-2027

O NeoCortex segue uma trilha de industrialização estruturada, migrando de um core estável em Python para uma camada de transporte de alta performance baseada em Rust.

---

## 📅 Cronograma

### Q2 2026: Fundação e Estabilização
- [x] **NeoCortex Core MVP:** Implementação completa da memória fractal e IPC mmap em Python.
- [x] **Governança Zero-Trust:** Execução criptográfica de ferramentas e logs de auditoria.
- [ ] **Teste Alpha:** Testes privados das 17 ferramentas principais.

### Q3 2026: Transição Open-Source
- [ ] **GitHub Milestone:** Lançamento público do framework sob Licença MIT.
- [ ] **Documentação Bilíngue:** Recursos completos para desenvolvedores em Inglês e Português.
- [ ] **Paper Acadêmico:** Publicação do estudo empírico "Arquitetura de Memória Fractal" no arXiv.

### Q4 2026: Migração Rust (Integração SOTA)
- [ ] **neocortex-mcp-rs:** Desenvolvimento do gateway nativo em Rust para latência de transporte sub-10ms.
- [ ] **Mapeamento O(1):** Portabilidade da lógica IPC do PicoClaw para Rust puro.
- [ ] **Enterprise Features:** Suporte multi-tenant e telemetria avançada.

### 2027: Expansão do Ecossistema
- [ ] **Marketplace de Integrações:** Pacotes de ferramentas criados pela comunidade para indústrias especializadas.
- [ ] **Managed Cloud Bridge:** Conectividade híbrida segura entre MCP local e fluxos em nuvem.
- [ ] **Parcerias Acadêmicas:** Projetos de pesquisa conjuntos com laboratórios de ciência da computação de elite.
